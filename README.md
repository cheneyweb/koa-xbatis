# koa-xbatis
极简风格的RESTful无后端Node框架，基于koa-xbatis中间件，只需要写SQL，然后直接RESTful请求，全自动CRUD

框架目录结构
>
	├── app.js
	├── config
	│   ├── default.json
	│   ├── develop.json
	│   └── production.json
	├── node_modules
	├── package.json
	├── src
	│   ├── nodebatis
	│   └── yaml
	└── xbatis_modules
	    └── koa-xbatis
